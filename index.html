My Simple API Documentation

Endpoint Description Method
/allBooks Get all books GET
/create Create a new book POST
/book/id Get book by ID GET
/update/id Update book by ID PUT
/delete/id Delete book by ID DELETE

To use in Postman simplified

Postman Description Info Method
http://localhost:3000/allBooks Get all books Choose: Body, Raw, JSON. GET
http://localhost:3000/create Create book -- Body: {"title": "ExampleTitle", "author":"ExampleAuthor"} Choose: Body, Raw, JSON, enter description body and enter the information of the book you want to add POST
http://localhost:3000/book/id Gets the book with a specific id. Change out the "id" in the URL to get a specific book by ID. GET
http://localhost:3000/update/id Updates the book with the choosen id. -- Body; {"title": "ExampleTitle", "author":"ExampleAuthor"} Change out the "id" in the URL to indicate what book you want to update, enter the description body and update title or author. PUT
http://localhost:3000/delete/id Removes a book by the choosen id from the database. Change out the "id" in the URL to indicate what book you want to update. DELETE
